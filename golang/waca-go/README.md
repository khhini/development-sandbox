# Waca

My Personal Bookmarks Systems.

# Feature

[] Basic CRUD: create, read, update and delete bookmarks
[] Metadata Extractions: extrating bookmarks metadata e.g., title, description, etc. ( extract via Open Graph Protocol, `title` & `h1` tags)
[] Semantic Search: using vector embeddings
[] AI Auto-Tagging: Use an LLM API (like Gemini or OpenAPI) to analyze the scraped content and suggest 3-5 relevant tags automatically.
[] AI Summarization: Generate a 3-bullet point "TL;DR" for every saved article to help users decide what to read first.
[] Broken Link Checker: automated job that flags 404s or moved the content in the user's library.
[] Reader Mode: A dedicated view that renders extracted article text in a clean, customizable layout.
